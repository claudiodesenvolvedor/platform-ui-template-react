# Platform UI Template

Plataforma frontend corporativa baseada em React + TypeScript.

Este repositório representa o **padrão oficial de frontend** da empresa e deve ser utilizado como base para todos os novos sistemas.

---

## 📌 Visão Geral

A Platform UI Template foi criada para resolver problemas comuns em projetos frontend corporativos:

- falta de padronização visual
- arquiteturas inconsistentes
- retrabalho entre projetos
- dificuldade de manutenção
- onboarding lento de novos desenvolvedores

Esta plataforma fornece uma fundação única, escalável e documentada.

---

## 🎯 Objetivos

- Padronizar layout, componentes e fluxo de navegação
- Centralizar identidade visual através de Design Tokens + Brand Config
- Reduzir tempo de criação de novos sistemas
- Facilitar manutenção e evolução
- Preparar o frontend para mudanças futuras de marca/gestão
- Criar uma base sólida para crescimento da empresa

---

## 🧱 Stack Tecnológica

- React + Vite + TypeScript
- React Router
- Axios
- Context API
- CSS puro com variáveis globais
- Design Tokens + Multi-brand
- Autenticação com rotas protegidas
- Sidebar por permissão
- Toast global
- Loader global
- Responsividade mínima corporativa

---

## 📁 Estrutura do Projeto

docs/ → documentação oficial da plataforma
public/ → arquivos estáticos
src/
layouts/ → layout corporativo (header + sidebar + content)
pages/ → telas do sistema
components → componentes reutilizáveis
theme/ → design tokens + brand config
hooks/ → contextos globais (auth, toast, loader)
services/ → integração com backend
styles/ → CSS centralizado
